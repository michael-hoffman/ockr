Built for flow

Think an Obsidian-style linked knowledge base, but native to Typst instead of Markdown — so every note is a real typesetting document with math, tables, and templates, edited the way you'd edit code.

Modal editing

Full Helix grammar — select-then-act motions, text objects, multi-cursor, counts, macros, a jump list. A Standard (VS Code-style) mode is one toggle away.

Live Typst preview ⌘⌥H

Every keystroke recompiles in the background. HTML for speed, paged for fidelity, PDF export with ⌘⇧E.

Wikilinks + graph ⌘⇧G

[[note name]] with autocomplete, backlinks per note, and an interactive graph of the whole vault.

Language intelligence

Optional tinymist LSP: diagnostics in the gutter, hover docs on K, go-to-definition on gd, completions on ⌃Space.

Fuzzy everything ⌃P

Path-based file picker, title-based quick switch (⌘K), command palette (⌘P), vault-wide search (⌘⇧F).

Plugins

Sandboxed WASM plugins with typed capability declarations — network, filesystem, console, and Typst packages.

Zen mode ⌃⌘Z

Sidebar and preview vanish; your writing column centres at a comfortable measure. Typewriter scrolling optional.

Plain files, no lock-in

A vault is just a folder of .typ files. Tabs, undo history, and settings persist locally. Nothing leaves your machine.

Install

Download the latest .dmg, open it, drag ockr to Applications. The app is ad-hoc signed — on first launch, right-click → Open → Open. All versions are on the releases page.

Build from source (Rust 1.96+, macOS 11+):

# clone, then:
cargo run --release
# or produce a distributable .app + .dmg:
scripts/bundle.sh

Optional LSP: install tinymist and put it on your PATH for diagnostics, hover, and go-to-definition. ockr runs fine without it.
